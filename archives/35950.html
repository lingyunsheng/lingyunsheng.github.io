一键全站变灰

置顶 | 发表于 2020-12-02
阅读次数： Valine：
本文字数： 2.8k 阅读时长 ≈ 3 分钟

很多网站包括我公司在内,重大节日或事故内默哀哀悼日,访问一些网站,全站都会变成灰色, 如果想到要立即开发,设计图修改耗费大量事件精力,这儿使用css3的filter滤镜,filter的grayscale属性

filter:grayscale 可以调整元素的灰度值

.filter-gray {
    /*  调整网站灰度值100% 兼容ie */
    filter: grayscale(100%);
    -webkit-filter: grayscale(100%);
    -moz-filter: grayscale(100%);
    -ms-filter: grayscale(100%);
    -o-filter: grayscale(100%);
    filter: url("data:image/svg+xml;utf8,#grayscale");
    filter: progid:DXImageTransform.Microsoft.BasicImage(grayscale=1);
    -webkit-filter:grayscale(1);
}

效果显示如下

正常情况下,导航栏是有一个吸顶效果的,鼠标无论怎么滑动,导航栏都固定在顶部,设置完一键全站变灰,
在css的body设置,导航栏没用固定在顶部而是穿透过屏幕外边了

为什么会发生这种情况呢

对于指定了filter样式值不为none时,被应用改样式的元素其子元素中如果有position为absolute和fixed的元素,会为其创建一个新的容器,使得这些元素定位于新创建的容器

解决方法

fixed是相对于html根元素定的,如果body设置了filter属性而且不为none的时候,body会创建一个新的容器,使用fixed定位的元素则相对于新容器,所以body的元素会滚出屏幕外

因此,将filter的grayscale设置在根元素html,这样html创建新的定位基准元素,也不会对子孙元素产生滚动到页面外的影响

html {
    filter: grayscale(100%);
    -webkit-filter: grayscale(100%);
    -moz-filter: grayscale(100%);
    -ms-filter: grayscale(100%);
    -o-filter: grayscale(100%);
    filter: url("data:image/svg+xml;utf8,#grayscale"); /*慎用 移动端和safari会失效*/
    filter: progid:DXImageTransform.Microsoft.BasicImage(grayscale=1);
    -webkit-filter: grayscale(1);
    filter:gray; /**ie6-9*/
}

修改后的效果

解决方案2

在body里增加一个gray-scale的容器 非全站变灰我们可以将需要使用filter的元素单独加上



<html>
    <body>
        <div class="gray-filter"></div>
    </body>
</html>

<style>
.fixed {
    position: fixed;
    top: 100px;
    left: 100px;
    height: 100px;
    width: 100px;
    background-color: #f00;
}
.gray-filter {
    filter: grayscale(100%);
    -webkit-filter: grayscale(100%);
    -moz-filter: grayscale(100%);
    -ms-filter: grayscale(100%);
    -o-filter: grayscale(100%);
    filter: url("data:image/svg+xml;utf8,<svg xmlns=\'http://www.w3.org/2000/svg\'><filter id=\'grayscale\'><feColorMatrix type=\'matrix\' values=\'0.3333 0.3333 0.3333 0 0 0.3333 0.3333 0.3333 0 0 0.3333 0.3333 0.3333 0 0 0 0 0 1 0\'/></filter></svg>#grayscale");
    filter: progid:DXImageTransform.Microsoft.BasicImage(grayscale=1);
    -webkit-filter: grayscale(1);  /***/
}
</style>

解决方案3

fixed定位元素 直接加css穿透事件,pointer-events:all; css属性,让遮罩层的所有事件都传递到下一层

兼容IE浏览器

兼容IE浏览器,引入grayscale.js

原理: js将color\background-color border-color等属性提取出来替换成灰色 而背景图片会使用canvas绘制处理成灰色再替换图片源

demo里的判断浏览器的userAgent来识别浏览器类型与版本再进行处理

grayscale.js处理 图片增加透明度
